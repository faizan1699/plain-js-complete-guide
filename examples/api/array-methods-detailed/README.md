# Detailed Array Methods Examples

This directory contains comprehensive examples for JavaScript array methods.

## Files

1. **01-map-detailed.js** - `Array.prototype.map`
   - Basic transformations
   - Object property extraction
   - Using index
   - Chaining with other methods
   - Edge cases

2. **02-filter-detailed.js** - `Array.prototype.filter`
   - Conditional filtering
   - Object filtering
   - Multiple conditions
   - Removing duplicates
   - Chaining patterns

3. **03-reduce-detailed.js** - `Array.prototype.reduce`
   - Summing values
   - Counting occurrences
   - Grouping data
   - Flattening arrays
   - Functional composition

4. **04-forEach-detailed.js** - `Array.prototype.forEach`
   - Basic iteration
   - Side effects
   - DOM manipulation
   - Performance considerations

5. **05-find-detailed.js** - `Array.prototype.find`
   - Finding first match
   - Object searching
   - Complex conditions
   - Difference from filter

6. **06-some-every-detailed.js** - `Array.prototype.some` & `every`
   - Boolean checks
   - Validation patterns
   - Short-circuit behavior
   - Comparison between methods

7. **07-slice-splice-detailed.js** - `Array.prototype.slice` & `splice`
   - Extracting portions
   - Removing elements
   - Inserting elements
   - Replacing elements
   - Non-mutating vs mutating

8. **08-sort-detailed.js** - `Array.prototype.sort`
   - Number sorting
   - String sorting
   - Object sorting
   - Multiple criteria
   - Common pitfalls

9. **09-flat-flatMap-detailed.js** - `Array.prototype.flat` & `flatMap`
   - Flattening nested arrays
   - Depth control
   - Map and flatten
   - Performance tips

10. **10-concat-includes-detailed.js** - `Array.prototype.concat` & `includes`
    - Combining arrays
    - Value checking
    - Index-based search
    - Modern alternatives

## Usage

Each file can be:
1. **Viewed directly** - Open in editor to see examples
2. **Referenced in documentation** - Use `data-src` attribute
3. **Run in console** - Copy code to browser console
4. **Used as reference** - Learn patterns and best practices

## Example Reference Format

```html
<code data-src="examples/api/array-methods-detailed/01-map-detailed.js"></code>
```
